Changelog
=========

1.0.0 (2024-04-22)
-------------------

First public release.

- Full Lavalink v4 WebSocket + REST support (v3 backwards-compatible)
- Multi-library support: discord.py, nextcord, disnake, py-cord
- Node strategy system: SHARD, LOCATION, USAGE, RANDOM
- Voice region routing
- Stackable named audio filters
- Session resuming with exponential backoff
- Node transfer with full state preservation
- IP route planner API
- Pyright strict typing throughout
- Optional orjson speedups
