WaveCord
========

.. image:: https://img.shields.io/pypi/v/wavecord
   :target: https://pypi.org/project/wavecord
.. image:: https://img.shields.io/pypi/pyversions/wavecord
   :target: https://pypi.org/project/wavecord
.. image:: https://img.shields.io/pypi/l/wavecord
   :target: https://github.com/Crowby-HQ/wavecord/blob/main/LICENSE

A modern, async-first **Lavalink v4** client for Python Discord libraries.

Works with **discord.py**, **nextcord**, **disnake** and **py-cord** out of the box.
