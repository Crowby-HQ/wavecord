# Security Policy

### Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.x     | ✅        |

### Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report them privately via [GitHub Security Advisories](https://github.com/Crowby-Studios/wavecord/security/advisories/new) or by emailing **hello@crowby.dev**.

We will respond within **72 hours** and aim to release a fix within **7 days** for critical issues.
